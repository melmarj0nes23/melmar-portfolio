# Profile Photos & Assets

Place your desired profile pictures, cover photos, or other portfolio images in this directory.

## How to upload and use your image:
1. **Upload**: Drag and drop your image file (e.g., `avatar.jpg`, `my-photo.png`, etc.) directly into this `/public/profile` folder using the file explorer on the left side of your AI Studio environment.
2. **Apply**: Enter the exact clean relative URL into the Avatar or Cover photo input in the app:
   - `/profile/avatar.jpg`
   - `/profile/my-photo.png`

*(Note: Ensure you write the name exactly as the file is spelled, including capital letters and space replacements).*
